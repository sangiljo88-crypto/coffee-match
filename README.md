# ☕ 커피 취향 월드컵 — 내 인생 커피 찾기

**🌐 라이브 주소: https://sangiljo88-crypto.github.io/coffee-match/**

카페24 원두커피 쇼핑몰 앞단에 붙이는 **인터랙티브 커피 추천 페이지**입니다.

손님이 이상형 월드컵처럼 **클릭 → 클릭 → 클릭** 12번의 선택을 하면,
**올데이 컬렉션 4종 + 대표 스페셜티 7종 = 커피 11종** 중 딱 맞는 커피를 추천하고
그 커피를 **어떤 형태(원두/드립백/드립에센스 + 용량)로 즐기면 좋은지**까지 함께 제안합니다.
**버튼 클릭 한 번에 카페24 상품 페이지로 이동**해 바로 결제할 수 있습니다.

## 테스트 흐름 (12번의 선택)

| 순서 | 화면 | 내용 |
|---|---|---|
| Q1 | **MBTI 선택** | 16유형 버튼 + "잘 몰라요" 건너뛰기 |
| Q2 | 나이 | 10~20대 / 30대 / 40대 / 50대 이상 |
| Q3 | **장소·용도** | 집·홈카페 / 사무실 / **카페·매장 납품** / 야외·여행 |
| Q4 | **시간대** | 아침 / 점심 / 오후 / 저녁 → 올데이 컬렉션 매칭 |

| Q7~Q12 | VS 월드컵 | 기구 보유 · 여유 시간 · 소비량 · 농도 · 블랙/라떼 · 핫/아이스 |

## 추천 커피 11종

**올데이 컬렉션** — 마시는 시간대에 맞춘 블렌드 (Q4 시간대 선택에 강한 가중치)

| 키 | 커피 | 맛 | 시간대 |
|---|---|---|---|
| `morningzest` | 모닝제스트 | 산미 | 아침 |
| `middayharmony` | 미드데이하모니 | 단맛 | 점심 |
| `afternoonbold` | 애프터눈볼드 | 쓴맛 | 오후 |
| `twilightamber` | 트와일라잇엠버 | 디카페인·군고구마 | 저녁 |

**대표 스페셜티** — 입맛 취향에 매칭

| 키 | 커피 | 맛 |
|---|---|---|
| `josangil` | 조상일 블렌드 | 밸런스 (동점 시 우선 추천되는 시그니처) |
| `ethiopia` | 에티오피아 아리차 에이미 내추럴 G1 | 산미 |
| `png` | 파푸아뉴기니 마라와카 블루마운틴 | 단맛 |
| `guatemala` | 과테말라 안티구아 산 로렌조 엘 쿠보 핀카 메디나 SHB | 쓴맛 |
| `kenya` | 케냐 마사이 AA | 산미+쓴맛 |
| `colombia` | 콜롬비아 후일라 수프리모 | 고소함+바디 |
| `brazil` | 브라질 미나스 제라이스 | 고소함 |

> 트와일라잇엠버는 올데이 컬렉션이자 유일한 디카페인이라, "저녁" 선택 시 가장 강하게 추천됩니다.

**추천 로직**: 커피 점수와 형태 점수를 따로 합산합니다.
- **커피**: 시간대(+3~4) + 입맛(산미/단맛/쓴맛/고소, +2) + 농도·메뉴·온도·MBTI·나이(+1)의 합계 1위 → 2·3위도 함께 노출
- **형태**: 장소·기구·여유시간·소비량 등으로 원두 200g/500g/1kg+, 드립백 12g/20g, 에센스 500ml/1L+ 중 1위를 "이렇게 즐겨보세요"로 제안
- "카페·매장 납품"을 고르면 형태 제안이 B2B(에센스 1L+ ↔ 그라인더 보유 시 원두 1kg+)로 확실히 라우팅됩니다

## ⭐ 내 상품 연결하기 (필수 설정)

`index.html`을 메모장/에디터로 열고 `★ 여기만 수정하면 됩니다` 라고 표시된 세 곳만 고치면 됩니다.
① `CONFIG`(쇼핑몰 이름·주소) ② `COFFEES`(커피 11종 — 이름·설명·**상품 URL**) ③ `FORMS`(형태 7종 — 링크는 선택사항)

### ① 쇼핑몰 기본 정보

```js
const CONFIG = {
  shopName: "BEANS COFFEE ROASTERS",        // ← 쇼핑몰 이름
  shopUrl:  "https://YOUR-SHOP.cafe24.com", // ← 카페24 쇼핑몰 주소
};
```

### ② 커피 11종 (COFFEES)

```js
ethiopia: {
  collection: "대표 스페셜티",        // 결과 화면 상단 뱃지
  name: "에티오피아 아리차",          // 커피 이름
  notes: ["산미", "베리·플로럴", …],  // 특징 태그 (첫 번째가 맛 태그로 강조됨)
  desc: "…",                        // 추천 문구
  url: "https://YOUR-SHOP.cafe24.com/product/detail.html?product_no=306", // ← 실제 상품 URL
},
```

**카페24 상품 URL 찾는 법** — 쇼핑몰에서 해당 상품 상세페이지를 연 뒤 주소창의 URL을 그대로 복사하세요.
보통 `https://내쇼핑몰.cafe24.com/product/상품명/123/` 또는 `…/product/detail.html?product_no=123` 형태입니다. 둘 다 사용 가능합니다.

### ③ 형태 7종 (FORMS)

결과 화면의 "이렇게 즐겨보세요" 제안 문구입니다. `url`을 채우면 링크로도 연결됩니다(빈칸이면 안내만 표시).

```js
essence1L: { name: "드립에센스 1L+", tip: "머신 없이 커피 메뉴 판매 — 카페·매장 납품용.", url: "" },
```

> 용량 옵션이 한 상품 페이지에 묶여 있다면(예: 200g/500g/1kg 선택 옵션), 같은 URL을 여러 곳에 넣어도 됩니다.

## 카페24 쇼핑몰에 붙이는 방법 (3가지 중 택 1)

### 방법 A. GitHub Pages 링크 (이미 설정 완료 ✨)

이 리포지토리는 GitHub Pages 자동 배포가 설정되어 있습니다.

- 라이브 주소: **https://sangiljo88-crypto.github.io/coffee-match/**
- `main` 브랜치에 변경 사항을 올리면 1~2분 안에 자동으로 사이트에 반영됩니다
- 카페24 관리자 → 메인 배너/팝업/메뉴에 이 주소를 링크하면 끝
  - 예: 메인에 "🏆 내 인생 커피 찾기" 배너 → 클릭 시 테스트 페이지 → 결과에서 상품 페이지로 복귀

결제는 원래 쇼핑몰에서 이뤄지므로 별도 연동 작업이 없습니다.

### 방법 B. 카페24 디자인 편집에 직접 넣기

1. 카페24 관리자 → **디자인(PC/모바일) → 디자인 편집**
2. 새 화면(페이지) 추가 → HTML 편집창에 `index.html` 내용 **전체 복사-붙여넣기**
3. 쇼핑몰 메뉴/배너에서 해당 페이지로 링크

### 방법 C. iframe으로 삽입

카페24 페이지에 아래 한 줄만 넣기:

```html
<iframe src="https://sangiljo88-crypto.github.io/coffee-match/" style="width:100%;height:100vh;border:0"></iframe>
```

## 질문·점수 커스터마이징

### 일반 질문 — `QUESTIONS` 배열

```js
{
  cat: "시간대",                          // 카테고리 뱃지
  q: "커피가 가장 당기는 시간은?",          // 질문
  options: [
    { emoji: "🌅", title: "아침", desc: "눈 뜨자마자 모닝커피",
      coffee: { morningzest: 3, ethiopia: 1, kenya: 1 },   // 커피 점수
      form: { essence500: 1, dripbag12: 1 },               // 형태 점수
      reason: "아침 첫 잔 루틴" },          // 결과 화면 "당신의 선택" 문구
  ],
}
```

- `coffee`는 커피 추천에, `form`은 "이렇게 즐겨보세요" 형태 제안에 각각 점수가 쌓입니다.
- 질문을 추가/삭제하면 진행 도장판 칸 수도 자동으로 맞춰집니다.
- `layout: "grid"`는 4지선다, `layout: "mbti"`는 MBTI 화면, 없으면 VS 2지선다입니다.

### MBTI 가중치 — `MBTI_LETTER_POINTS`

E/I/S/N/T/F/J/P 각 글자가 어떤 커피·형태에 1점을 주는지 정의합니다. 예:

```js
N: { coffee: { ethiopia: 1 }, form: { essence500: 1, dripbag20: 1 } },  // 새로운 것에 호기심
```

색상을 바꾸려면 CSS 상단의 `:root { --stamp: … }` 변수만 수정하면 됩니다.
