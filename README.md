# ☕ 커피 취향 월드컵 — 내 인생 커피 찾기

**🌐 라이브 주소: https://sangiljo88-crypto.github.io/coffee-match/**

카페24 원두커피 쇼핑몰 앞단에 붙이는 **인터랙티브 커피 추천 페이지**입니다.

손님이 이상형 월드컵처럼 **클릭 → 클릭 → 클릭** 10번의 선택을 하면,
**원두커피 / 드립백커피 / 드립에센스** 3개 라인 · 7종 제품 중 딱 맞는 것을 추천하고
**버튼 클릭 한 번에 카페24 상품 페이지로 이동**해 바로 결제할 수 있습니다.

## 테스트 흐름 (10번의 선택)

| 순서 | 화면 | 내용 |
|---|---|---|
| Q1 | **MBTI 선택** | 16유형 버튼 + "잘 몰라요" 건너뛰기 |
| Q2 | 나이 | 10~20대 / 30대 / 40대 / 50대 이상 |
| Q3 | **장소·용도** | 집·홈카페 / 사무실 / **카페·매장 납품** / 야외·여행 |
| Q4~Q10 | VS 월드컵 | 기구 보유 · 여유 시간 · 소비량 · 농도 · 블랙/라떼 · 쟁여두기/신선 · 핫/아이스 |

## 추천 제품 7종

| 키 | 제품 | 핵심 포인트 |
|---|---|---|
| `beans200` | 원두커피 200g | 그라인더·드립기구 보유자, 갓 볶은 신선함, 홈카페 입문 |
| `beans500` | 원두커피 500g | 매일 내리는 홈바리스타, 신선함·가성비 밸런스 |
| `beans1kg` | 원두커피 1kg+ | 헤비유저·업소용 대용량, g당 최고 가성비 |
| `dripbag12` | 드립백커피 12g | 기구 없이 물만 부으면 끝, 사무실·여행·캠핑 |
| `dripbag20` | 드립백커피 20g 대용량 | 전문점 핸드드립과 같은 원두량, 진한 풍미 |
| `essence500` | 드립에센스 500ml | 핸드드립 농축 원액, 더치와 달리 **따뜻하게도 맛있음**, 물/우유 부어 10초 완성, 라떼 가능 |
| `essence1L` | 드립에센스 1L+ | **카페 납품용** — 에스프레소 머신·그라인더 없이 커피 메뉴 판매 |

**추천 로직**: 선택마다 어울리는 제품에 점수가 쌓이고 최종 1위를 추천(2·3위도 함께 노출).
"카페·매장 납품"을 고르면 B2B 제품(에센스 1L+, 원두 1kg+)에 큰 가중치가 붙어 확실히 사업자용으로 안내되고,
그 안에서 "그라인더 있음 → 원두 1kg+" / "머신 없음·10초 제조 → 에센스 1L+"로 갈립니다.

`index.html`을 메모장/에디터로 열고 `★ 여기만 수정하면 됩니다` 라고 표시된 두 곳만 고치면 됩니다.

### ① 쇼핑몰 기본 정보

```js
const CONFIG = {
  shopName: "BEANS COFFEE ROASTERS",        // ← 쇼핑몰 이름
  shopUrl:  "https://YOUR-SHOP.cafe24.com", // ← 카페24 쇼핑몰 주소
};
```

### ② 제품 7종 (PRODUCTS)

```js
beans200: {
  name: "원두커피 200g",            // 상품명
  notes: ["그라인더·드립기구용", …],  // 특징 태그
  desc: "…",                       // 추천 문구
  url: "https://YOUR-SHOP.cafe24.com/product/detail.html?product_no=201", // ← 실제 상품 URL
},
```

**카페24 상품 URL 찾는 법** — 쇼핑몰에서 해당 상품 상세페이지를 연 뒤 주소창의 URL을 그대로 복사하세요.
보통 `https://내쇼핑몰.cafe24.com/product/상품명/123/` 또는 `…/product/detail.html?product_no=123` 형태입니다. 둘 다 사용 가능합니다.

> 용량 옵션이 한 상품 페이지에 묶여 있다면(예: 200g/500g/1kg 선택 옵션), 같은 URL을 여러 키에 넣어도 됩니다.

### 방법 A. GitHub Pages 링크 (이미 설정 완료 ✨)

이 리포지토리는 GitHub Pages 자동 배포가 설정되어 있습니다.

## 질문·점수 커스터마이징

### 일반 질문 — `QUESTIONS` 배열

```js
{
  cat: "기구",                        // 카테고리 뱃지
  q: "커피 기구, 갖고 계세요?",         // 질문
  options: [
    { emoji: "⚙️", title: "그라인더·드립세트 있어요", desc: "장비 세팅 완료",
      points: { beans200: 2, beans500: 2, beans1kg: 1 },  // 이 선택 시 점수를 받는 제품
      reason: "홈카페 장비 보유" },                         // 결과 화면 "당신의 선택" 문구
  ],
}
```

- `points`의 숫자만큼 해당 제품에 점수가 쌓이고, 최종 합계 1위가 추천 제품이 됩니다.
- 질문을 추가/삭제하면 진행 도장판 칸 수도 자동으로 맞춰집니다.
- `layout: "grid"`는 4지선다, `layout: "mbti"`는 MBTI 화면, 없으면 VS 2지선다입니다.

### MBTI 가중치 — `MBTI_LETTER_POINTS`

E/I/S/N/T/F/J/P 각 글자가 어떤 제품에 1점을 주는지 정의합니다. 예:

```js
T: { essence500: 1, beans1kg: 1 },  // 효율·가성비 중시 → 10초 원액, 대용량
```

색상을 바꾸려면 CSS 상단의 `:root { --stamp: … }` 변수만 수정하면 됩니다.
